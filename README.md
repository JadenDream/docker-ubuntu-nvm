# Docker Ubuntu with NVM
A Ubuntu based Docker with NVM

一個使用 ubuntu 基底的docker 提供 nvm 來編譯 nodejs 的環境(包含一些基本的服務以供便利編譯需求)

## Installing This Image
```
docker pull jaden0416/docker-ubuntu-nvm
```
